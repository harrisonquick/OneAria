# Aria People Seed Data

`ariaPeopleSeed.json` is synthetic data modeled from the saved Aria People profile page in this workspace.

The seed data intentionally keeps the real downloaded page out of source control and does not include real employee names, emails, phone numbers, Slack IDs, profile URLs, or organization IDs. It mirrors the page structure so the application can be built against realistic profile, organization, contact, location, badge, tag, link, and availability data.

Modeled UI areas:

- Profile header: name, title, location, email, phone, permalink, pronouns, profile photo state.
- About: summary text, empty-state prompts, badges and patents.
- Organization: manager chain, direct reports, org chart links.
- Links and tags: populated and empty states.
- Additional information: cost center, job title, company code, product association, HR contacts, Slack workspaces.
- Contact information: Slack/message actions and vCard download action.
- Location: address, building, workspace type, time zone, local time.
